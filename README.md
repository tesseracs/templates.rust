# Tesseracs — rust template

Starter project for [Tesseracs](https://github.com/tesseracs) chat sessions.

Clone URL: `https://github.com/tesseracs/templates.rust`

## Layout

- `src/main.rs` — binary; `use templates_rust::greet` from the library.
- `src/lib.rs` — library crate API (`pub fn greet`).
- `run.sh` — `cargo build --release` and run the binary from `target/release/`.
